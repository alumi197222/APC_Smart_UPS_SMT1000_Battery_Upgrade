# APC Smart UPS 改裝

SMC1000、SMT1000都與 SMT1500的外殼相同，只需將1000系列的內部稍加修改及即可使用1500系列電池。

## SMT1000 原廠電池規格

- 原廠使用電池組:RBC6 
- 2 × 12V 12Ah
- CSB GP12120 F2
- 尺寸 151(L)*98(W)*100(H) 單位mm

## SMT1000、SMC1000  電池替代品

## SMT1500 原廠電池規格

- 原廠使用電池組:RBC7
- 2 × 12V 18Ah
- CSB GP12170
- 尺寸 181(L)*76(W)*167(H) 單位mm
